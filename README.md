# BoxMate - Optimum USA Equipment Guide

BoxMate is a smart guide for Optimum USA agents handling unactivated cable boxes. When a customer receives a cable box from the store that isn't activated, agents use **Box Maintenance** to add the equipment to the corporate inventory. This tool streamlines the process by providing quick and accurate guidance.

## 🔹 Features
- Easy navigation for selecting equipment type, manufacturer, and status.
- Step-by-step guidance for adding devices via **Box Maintenance**.
- Designed for Optimum USA agents to improve efficiency.

## 🚀 How to Use
1. Navigate to [BoxMate](https://msmelok.github.io/box-mate/).
2. Select the equipment type and manufacturer.
3. Follow the provided steps to add the equipment.
4. Ensure the device is properly added to the corporate inventory.
